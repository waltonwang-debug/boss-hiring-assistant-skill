# 约面与飞书服务

## 目标

只负责：

- 在候选人已确认时间后创建飞书面试日程
- 把飞书日程链接发回候选人

不负责：

- 筛选简历
- 自由聊天

## 进入条件

只有在下面条件同时满足时才进入：

1. 用户明确同意该候选人进入约面
2. 候选人已经确认可面试时间

## 飞书 bot 路径

唯一允许路径：

1. 用户自己去飞书开放平台创建应用或 bot
2. 用户自己获取 `App ID` / `App Secret`
3. 用户把凭证交给龙虾
4. 龙虾完成本地 bot 配置
5. 龙虾明确告诉用户去飞书 bot 聊天里测试创建一个日程
6. 用户在 bot 聊天里完成真正可用的授权
7. 后续龙虾再通过这个 bot 创建真实日程

## 禁止项

龙虾不得：

- 主动打开飞书开放平台网页
- 替用户点击开放平台后台
- 自由尝试 API / OAuth / 网页版

## 第一次创建日程时才确认的内容

第一次真的要创建日程时，再确认：

- bot 是否已配置
- bot 是否已在聊天里完成日历授权
- 默认参会人是只加用户自己，还是再加一个固定账号

## 失败处理

如果创建失败，先判断是否是 bot 权限或用户授权问题。
若是，告诉用户具体要去 bot 聊天里补哪一步，不要切换到其他实现路径。
